# rock-paper-scissors
A simple terminal Rock - Paper - Scissors game

How to play:
 User chooses Rock, Paper or Scissors. 
 The bot makes a choise too. (Randomly)
 
 Rock > Scissors
 Rock = Rock
 Rock < Paper
 .
 Paper > Rock
 Paper = Paper
 Paper < Scissors
 .
 Scissors > Paper
 Scissors = Scissors
 Scissors < Rock
 
 Very simple. Built using switch-cases.
